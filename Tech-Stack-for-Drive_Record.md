**เหตุผลที่เลือก :** เบา เร็ว มี Extension ครอบจักรวาล และนักพัฒนาทั่วโลกใช้งานเป็นหลัก

**ใช้ฟรีหรือไม่ :** ฟรี 100% (Open Source)

**ควรติดตั้งช่วงไหนของโปรเจกต์ :** ก่อนเริ่มโปรเจกต์

**จำเป็นหรือไม่ :** จำเป็น

# Database GUI

**ชื่อเครื่องมือ :** Supabase Studio

**ใช้ทำอะไร :** หน้า Dashboard สำหรับดู แก้ไข และจัดการข้อมูลในฐานข้อมูลผ่านเบราว์เซอร์

**เหตุผลที่เลือก :** มีมาให้พร้อมกับโปรเจกต์ Supabase ทันที ใช้งานง่าย ไม่ต้องติดตั้งโปรแกรมในเครื่องเพิ่ม

**ใช้ฟรีหรือไม่ :** ฟรี (รวมอยู่ใน Supabase)

**ควรติดตั้งช่วงไหนของโปรเจกต์ :** Phase 3

**จำเป็นหรือไม่ :** จำเป็น

# AI Tools

**ชื่อเครื่องมือ :** GitHub Copilot

**ใช้ทำอะไร :** เป็นผู้ช่วย AI ฝังอยู่ใน VS Code ช่วยแนะนำและเขียนโค้ดให้แบบ Real-time

**เหตุผลที่เลือก :** เข้าใจบริบทของโปรเจกต์ได้ดีมาก ช่วยเขียน Boilerplate หรือฟังก์ชันซ้ำซากได้เร็วขึ้น 30-50%

**ใช้ฟรีหรือไม่ :** มีค่าใช้จ่าย (แต่คุ้มค่าที่สุดสำหรับการลดเวลาพัฒนา)

**ควรติดตั้งช่วงไหนของโปรเจกต์ :** ก่อนเริ่มโปรเจกต์

**จำเป็นหรือไม่ :** ไม่จำเป็น (แต่แนะนำอย่างยิ่งสำหรับ Solo Developer)
